# Implementation Prompt Archive

This archive keeps the final high-level prompts used to guide the project. Historical working notes were condensed for the final GitHub submission.

## Final Project Prompt
Build a small GenAI app, skill, or tool for one narrow business use case. The artifact must be runnable, documented, evaluated against a simpler baseline, and packaged in a GitHub repository with clear setup instructions.

## TokenLess Product Prompt
Create a chat-first prompt mentor for software developers using vibe-coding tools. The system should collect missing project details, generate a structured prompt, compare it against a raw-prompt baseline, and explain where the workflow succeeds and fails.

## Engineering Prompt
Keep the design simple and reproducible. Use local LM Studio for planning and optimization, cloud judges for evaluation, Streamlit for the final app, offline tests for guardrails, and a README that lets a grader install and run at least one example.
